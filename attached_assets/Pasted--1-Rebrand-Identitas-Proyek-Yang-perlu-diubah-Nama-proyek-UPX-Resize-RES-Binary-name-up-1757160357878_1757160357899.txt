Banner / Help / Output CLI: semua teks "UPX" → "RESIZE"

📂 2. Struktur & Nama File

CHANGELOG baru: catat perubahan besar (branding, perbaikan parser, CLI, dsb).

-DUPX_VERSION → -DRES_VERSION
